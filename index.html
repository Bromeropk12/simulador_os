Simulador Round Robin

Algoritmo de planificación de procesos - Fácil de Entender

Ingreso de Procesos

Agrega procesos manualmente o carga ejemplos predefinidos. Cada proceso tiene un nombre y un tiempo de ejecución (burst time).

Configuración

Define el quantum: tiempo máximo que un proceso puede usar la CPU antes de ser interrumpido.

Simulación

Inicia la simulación automática o usa el modo paso a paso para ver cada cambio manualmente.

Visualización

Observa el estado de la CPU y la cola de procesos esperando.

Diagrama de Gantt

Muestra cada periodo de ejecución de los procesos en orden cronológico. Cada barra horizontal indica un proceso y su duración exacta en segundos durante ese turno.

Resultados

Tabla con métricas finales: tiempo de ejecución original, tiempo de espera y tiempo de retorno para cada proceso.
